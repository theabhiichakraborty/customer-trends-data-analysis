# Customer Behavior Data Analysis Portfolio Project

## 📌 Project Overview
This project represents a comprehensive, end-to-end data analytics workflow. It is designed to analyze retail customer data to uncover insights regarding customer segmentation, purchasing trends, and loyalty.

**Author:** Shreyas  
**Institution:** NIT Silchar  

The goal of this project is to translate raw data into strategic business intelligence by:
- **Data Preparation & Exploratory Data Analysis (Python):** Cleaning and transforming the raw dataset using pandas.
- **Data Analysis (SQL):** Running queries to extract insights on customer segments, loyalty, and purchase drivers.
- **Visualization (Power BI):** Building interactive dashboards to highlight key patterns and trends for stakeholders.

## 🛠️ Tech Stack Used
- **Python:** For data cleaning and exploratory data analysis (Pandas, Jupyter Notebook).
- **SQL (PostgreSQL / MySQL):** For data modeling and answering business questions.
- **Power BI:** For building the final interactive dashboard.

## 🚀 How to Run the Project

1. **Clone the repository**
   ```bash
   git clone https://github.com/theabhiichakraborty/customer-trends-data-analysis.git
   cd customer-trends-data-analysis
   ```

2. **Run the Jupyter Notebook**
   Open `Customer_Shopping_Behavior_Analysis.ipynb` to see the data cleaning and EDA process.

3. **Database Setup & SQL Analysis**
   - Import the cleaned CSV file into your SQL database.
   - Run the queries provided in `customer_behavior_sql_queries.sql` to answer the key business questions.

4. **Power BI Dashboard**
   Open `customer_behavior_dashboard.pbix` in Power BI Desktop to interact with the visualizations.

## 📈 Key Business Questions Addressed
- What is the total revenue generated by male vs. female customers?
- Which products have the highest average review ratings?
- Do subscribed customers spend more compared to non-subscribers?
- Customer segmentation (New, Returning, Loyal) based on purchase history.
- Revenue contribution by different age groups.
